# samsung-logistics-dashboard
Power BI dashboard project for Samsung supply chain and logistics analysis using multiple CSV datasets.



# Samsung Supply Chain & Logistics Power BI Dashboard

## Project Objective
The objective of this project is to analyze Samsung’s supply chain and logistics operations using Power BI dashboards and business intelligence techniques. The project helps monitor procurement, inventory, production, shipments, and sales performance to support better operational decision-making.

---

# Business Questions Solved

- How efficiently is inventory being managed?
- Which suppliers are performing best?
- What are the production and shipment trends?
- Which products generate maximum sales?
- How can logistics delays and inventory issues be identified?
- Which facilities contribute most to operations?
- How does procurement impact supply chain performance?

---

# Detailed Description of Data

The project uses multiple dimension and fact tables to simulate a real-world supply chain database.

## Dimension Tables
- `dim_customer.csv`
- `dim_date.csv`
- `dim_facility.csv`
- `dim_product.csv`
- `dim_supplier.csv`

## Fact Tables
- `fact_inventory.csv`
- `fact_procurement.csv`
- `fact_production.csv`
- `fact_sales.csv`
- `fact_shipment.csv`

These datasets contain information related to customers, suppliers, products, facilities, inventory levels, procurement activities, production records, shipment tracking, and sales transactions.

# Detailed Steps Followed

1. Imported multiple CSV datasets into Power BI.
2. Performed data cleaning and preprocessing.
3. Created relationships between dimension and fact tables.
4. Developed a star schema data model.
5. Created calculated columns and DAX measures.
6. Built KPI cards and interactive visualizations.
7. Designed dashboards for supply chain insights.
8. Added filters and slicers for dynamic analysis.
9. Analyzed operational performance and business metrics.

---

# Challenges Faced

- Managing relationships between multiple datasets.
- Handling missing and inconsistent data.
- Optimizing dashboard performance.
- Creating accurate DAX calculations.
- Designing meaningful KPIs for supply chain analysis.
