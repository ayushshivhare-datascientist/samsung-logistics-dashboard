# samsung-logistics-dashboard
Power BI dashboard project for Samsung supply chain and logistics analysis using multiple CSV datasets.


  # Samsung Supply Chain & Logistics Power BI Dashboard

## Project Overview
This project presents a Supply Chain and Logistics Dashboard developed in Power BI using Samsung supply chain datasets. The dashboard provides insights into procurement, inventory, production, shipments, and sales performance.

## Tools & Technologies
- Power BI
- CSV Datasets
- Data Modeling
- DAX
- Data Visualization

## Dataset Files
- Customer Data
- Product Data
- Supplier Data
- Inventory Data
- Procurement Data
- Production Data
- Shipment Data
- Sales Data

## Dashboard Features
- Inventory Analysis
- Shipment Tracking
- Supplier Performance
- Sales Monitoring
- Production Insights
- Logistics KPIs

## Objective
To analyze and visualize supply chain operations and logistics performance using Power BI dashboards and business intelligence techniques.
